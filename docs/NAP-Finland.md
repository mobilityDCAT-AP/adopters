## Access metadata

By opening the page of a dataset (e.g., dataset at this [link](https://catalog.digitraffic.fi/en/dataset/e05167e6-47d8-4787-993e-fa0ae0bf2e91)), it is possible to download a representation of metadata in JSON-LD, N3, Turtle, and RDF/XML compliant with mobilityDCAT-AP.

The metadata for the whole catalog in different formats is also available:
- https://catalog.digitraffic.fi/en/catalog.jsonld
- https://catalog.digitraffic.fi/en/catalog.xml
- https://catalog.digitraffic.fi/en/catalog.ttl
- https://catalog.digitraffic.fi/en/catalog.n3
