## Access metadata

By openining the page of a dataset, it is possible to manually download a representation of metadata in JSON-LD,Turtle, and RDF/XML compliant with mobilityDCAT-AP and other metadata specifications.


## Harvest metadata

Metadata of individual datasets, or even of the entire catalogue, can be accessed via a harvesting API. This API is explained in the [FAQ](https://mobilithek.info/help/faq), see section "Harvesting API".
